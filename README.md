AI-Enabled Cryptocurrency Portfolio Management System with Real-Time Risk & Scam Detection

Built during Infosys Springboard Internship 6.0 (Nov 2025 – Jan 2026)

## Tech Stack
Spring Boot · React.js · MySQL · JWT Authentication · REST APIs

## APIs Used
CoinGecko · Binance · Etherscan

## Features
- Real-time crypto price tracking
- P&L calculation per holding
- Scam/risk token detection using blockchain intelligence
- JWT-secured user authentication
- Interactive dashboard
